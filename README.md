# Trigonometric_calculator
Chongqing University software Engineering course assignment

>1 程序简介  
    本程序实现了一个三角函数误差均值计算，能够通过输入数值和计算指令，程序内部完成三角函数计算后显示误差均值结果。  

>2 主要功能简介
>>2.1 sin函数误差均值计算
    输入：计算数值
    输出：误差均值结果
    功能：完成sin函数误差均值计算
    
>>2.2 cos函数误差均值计算
    输入：计算数值
    输出：误差均值结果
    功能：完成cos函数误差均值计算
    
>>2.3 arcsin函数误差均值计算
    输入：计算数值
    输出：误差均值结果
    功能：完成arcsin函数误差均值计算
    
>>2.4 arctan函数误差均值计算
    输入：计算数值
    输出：误差均值结果
    功能：完成arctan函数误差均值计算
